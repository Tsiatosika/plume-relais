# Plume Relais

Application d'écriture collaborative React Native / Expo.

## Stack
- React Native + Expo Router
- Supabase (auth, database, realtime, storage)
- Zustand (state management)

## Lancer le projet
npm install
npx expo start --web

## Fonctionnalités
- Authentification email/mot de passe
- Création et gestion d'histoires
- Mode aveugle
- Propositions et votes en temps réel
- Clôture de tour automatique
- Profil et réputation
